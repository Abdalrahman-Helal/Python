# Init_GPIO

## Task
Write Python code to generate an AVR GPIO init function and save it in a C file.

## What the script does
- Asks for a port name.
- Asks for the mode of each bit (`in` or `out`).
- Builds the DDR value.
- Writes the generated function to `Init_GPIO.c`.

## Run
```bash
python Init_GPIO.py
```
