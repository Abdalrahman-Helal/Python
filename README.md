# Python

Each folder contains a small Python application or task, along with a markdown note and the task output.

Some of these tasks also demonstrate Python file handling, such as reading from or writing to text files.

## Folders

- [CLI_Arg/](CLI_Arg/) - command-line arguments task.
- [count_lines/](count_lines/) - count lines in a text file.
- [count_words/](count_words/) - count words in a text file.
- [list_file/](list_file/) - write a Python list to a file.
- [Init_GPIO/](Init_GPIO/) - generate an AVR GPIO init function.
